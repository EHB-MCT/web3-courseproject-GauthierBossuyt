Recommender System

What is a recommender system?

A recommender system seeks to predict what a user might like based on collected information.

Types of recommender systems:

  • Popularity

    Recommends songs based of the song's popularity. The songs popularity depends on the metric used (time, user interactions, user ratings, ...).

  • Content-based

    A content-based recommender uses attributes from items the user interacted with to recommend similar items. The TF-IDF is a common method used for this type of recommender.

  • Collaborative

    Uses collaborative filtering to recommend items based of interactions by similar users.

  • Hybrid

    A combination of the collaborative and content-based recommender. The hybrid recommender performs better, because it uses high variance and enables variety and weighting.

Types of filtering:

  • Collaborative filtering

    Uses data from people, who have similar taste to make predictions about what you might like. Spotify uses this type of filtering to create an user's Discover Weekly playlist.

  • Natural Language Processing

    Using computers to process and analyze natural language data to automaticly summarize the main points of a text document. Spotify's NLP converts playlist to text documents to analyze how lyrical patterns relate to each other. Spotify's NLP tries to locate places where the term is used and tries to predict the meaning of the context. For example, when using the word 'ABBA' 'Dancing Queen' has a high score with the adjectives 'perky' and 'nonviolent'.

  • Audio Models

    Processing raw audio to create a range of characteristics, such as key, energy, tempo, ... . This way new songs from unknown artists can be recommended.

The diffence between explicit and implicit feedback:

  • Explicit

    Explicit feedback is direct and quantitative data collected from users. Explicit feedback is rare, because rating items isn't done frequently.

  • Implicit

    Implicit feedback is indirect and made from a users interactions. Implicit feedback comes in quite large quantities and allows to make recommendations in real time. Nowadays most recommender systems are built with implicit feedback.

Metrics to evaluate a recommender system:

  • Accuracy (for classification problems)

  • RMSE (for regression problems)

  • Hit Ratio @ 10
